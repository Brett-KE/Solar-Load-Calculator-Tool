What it does
Step 1 — Add Appliances

Displays a numbered catalogue of 15 common household appliances (fans, fridge, kettle, AC, water pump, etc.) plus a Custom option
For each appliance you enter: quantity and daily usage hours
Repeat until all loads are added

Step 2 — System Parameters

Asks for peak sun hours (defaults to 5.5 h — suitable for Kenya/East Africa)
Battery autonomy days, bank voltage (12/24/48 V)
Advanced: system efficiency %, depth-of-discharge %, safety/growth factor
